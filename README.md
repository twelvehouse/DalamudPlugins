# Dalamud Plugin Repository

Custom Dalamud plugin repository by [twelvehouse](https://github.com/twelvehouse).

## Setup

Add the following URL to **Dalamud → Settings → Experimental → Custom Plugin Repositories**:

```
https://raw.githubusercontent.com/twelvehouse/DalamudPlugins/main/pluginmaster.json
```

## Plugins

---

### ChatAnywhere &nbsp; `v0.1.17.1`

Chat in FFXIV from your browser anywhere.
Host your chat on a web server so you can talk from anywhere.

---

### FootIK &nbsp; `v1.1.0.0`

Foot IK for FFXIV — feet stay on the ground.
Real-time foot IK for FFXIV characters. Keeps feet planted naturally on terrain (stairs, slopes, uneven ground) during normal gameplay.
